# Swiggy Restaurant Data Analysis

## Project Overview
This project presents an end-to-end analysis of Swiggy restaurant data using SQL, Python, and Power BI. The objective is to clean, explore, analyze, and visualize restaurant data to identify useful insights related to restaurant prices, ratings, food types, cities, areas, and delivery time.

## Tools and Technologies
- SQL
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Jupyter Notebook

## Dataset
The dataset contains 8,680 restaurant records with 10 columns:

- ID
- Area
- City
- Restaurant
- Price
- Avg ratings
- Total ratings
- Food type
- Address
- Delivery time

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. SQL Analysis
4. Python Analysis
5. Data Visualization
6. Power BI Dashboard Creation

## Power BI Dashboard
An interactive Power BI dashboard was created to visualize key restaurant insights.

The dashboard includes:

- Total Restaurants
- Average Price
- Average Ratings
- City-wise Analysis
- Restaurant Analysis
- Food Type Analysis
- Rating Distribution
- Interactive Slicers

## Key Insights
- Comparison of restaurant prices across different cities
- Identification of highly rated restaurants
- Analysis of popular food categories
- Comparison of restaurant availability across cities
- Analysis of average restaurant prices
- Identification of rating and pricing patterns
